# Sprint Training Advisor

A rule-based Python program that recommends sprint, strength, recovery, and modified sessions from a date-based training progression.

## Features

- Calculates scheduled workouts from a season start date
- Supports tempo, acceleration, top-speed, endurance, and rest days
- Uses week-specific endurance workouts
- Provides modified sessions for hip, calf, hamstring, core, and quad soreness
- Includes recovery guidance, mobility, foot strength, and indoor/rainy-day options
- Logs completed workouts and notes to a CSV file
- Lets the user view, add, edit, and delete training-log entries through Python
- Supports schedule shifting after missed training days

## Requirements

- Python 3
- No external libraries required

## How to Run

Open Terminal in the project folder and run:

```bash
python3 sprint_training_advisor.py
